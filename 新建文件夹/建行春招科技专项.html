一、职业基本知识

根据岗位不同，专业题部分内容有所不同；

  1. 综合类笔试侧重考察经济学、财政金融学、货币银行学、会计学基础、法律、营销、管理、数理统计、信息技术等方面应知应会的知识；
  2. 信息技术类笔试侧重考察计算机网络、操作系统、软件工程、信息安全、设计模式、数据结构与算法、开发语言语法、数据库（语法）等方面应知应会的知识。

网络笔经：职业基础知识测试，包括计算机的一些知识（如数据结构、软件工程、数据库、计算机网络、操作系统、java、sql、计算机组成原理、区块链、网络安全等），占大部分，几十道的样子；建行行情行史，一两道或两三道的样子；二十大时政，两三道的样子；金融财政基础知识，四五道的样子等。

二、职业行为测试

考察时事政治、案例分析等

三、通用职业能力测验（一）

EPI（Employability Profile Inventory）：包括言语理解、逻辑推理、数量关系、资料分析这几大部分。在试卷中主要以单选题的形式出现，数字计算和推理题较多，比较注重对考生计算和逻辑推理能力的考察。EPI 部分的考试题一直都是银行考试的重点，能在短时间内提高成绩，所以建议大家把握好做题速度，抓住这部分得分点。

①言语理解

中国建设银行线上笔试之 言语作为人的一种思想表达方式，极为灵活多变。国考行测科目言语理解与表达板块旨在考查应聘者对于语言材料的理解能力以及借助语言准确表达的能力。这就需要应聘者从材料中准确把握作者的观点、态度、情感倾向。而作者的观点、态度、情感倾向等都会由一些特殊的关键词句引出。如表示总结的总结词、表示结果的因果词、表示转折的转折词等。在阅读题干时标出这些关键词能帮助我们快速定位文段的重点所在。

②数量关系

主要测查报考者理解、把握事物间量化关系和解决数量关系问题的能力，主要涉及数据关系的分析、推理、判断、运算等。

四、通用职业能力测验（二）

①逻辑推理

中国建设银行统一笔试之判断推理主要测查报考者对各种事物关系的分析推理能力，涉及对图形、语词概念、事物关系和文字材料的理解、比较、组合、演绎和归纳等。常见的题型有：图形推理、定义判断、类比推理、逻辑判断等

②资料分析

中国建设银行校招笔试之资料分析材料的来源： 资料分析的材料一般来自于国家统计局或者各个地方统计部门。 资料分析考查的能力： 考查考生对于给定材料的阅读、理解、分析及计算等能力。 资料分析难不难： 资料分析知识点比较少，也简单易懂。 资料分析包含哪些知识点： 资料分析主要包括现期量与基期量相关、增长率相关、增长量相关、比重相关，平均数与倍数相关等考点。

五、英语

英语考试形式大多是选词填空加阅读理解，单选主要是考察语法、词汇这类。阅读理解想必经过四六级的历练，这些对大家来说都不算啥，阅读理解难度都不算太大

六、性格测试

性格测试：不计入笔试成绩，只作为面试官评分的参考依据。性格测试主要是检测考生心里有无疾病以及考生的职业匹配度，一般只要考生真实作答，答案不是过于偏激，都可以通过。

    D 既可实现加密，也可实现数字签名

    本题考查计算机基础。

    对称秘钥加密算法是指采用单钥密码系统的加密方法，同一个密钥可以同时用作信息的加密和解密，也称为单密钥加密。非对称秘钥加密算法是指加密和解密使用不同密钥的加密算法，也称为公私钥加密。

    A 项错误，非对称密钥加密算法的强度复杂，安全性依赖于算法与密钥。但是由于其算法复杂，而使得加密解密的效率没有对称加密解密的效率高。

题目：某个数据通信系统采用 CRC 检验方式，并且生成多项式 G (x) 的二进制比特序列为 11001，目的节点接收到的二进制比特序列为 10111001 (含 CRC 检验码)，请判断传输过程中是否出现了差错？为什么？

通过发送方计算出检验码（发送方）：
W=D*2N （D 是发送数据，N 为检验码位数）
N 是几 就往后面补几个 0，然后得到的 W 对生成多项式 G 进行取余，得到的结果就是检验码的值

检验接收方判断是否出现差错（接收方）
通过目的节点接收到的含 CRC 检验码的二进制比特序列 W 对 生成多项式 G 进行取余，余数为 0 正确，否则出现差错

  9. 数据库一个事务执行过程中，其正在访问的数据被其他事务所修改，导致处理结果不正确，这是由于违背了事务的（ ）而引起的。

    A 原子性

    B 一致性

    C 隔离性

    D 持久性

    正确答案是： C

    解析

    本题考查的是数据库基础。事务的四大属性即事务的原子性、一致性、隔离性、持久性。事务的隔离性是多个用户并发访问数据库时，数据库为每一个用户开启的事务，不能被其他事务的操作数据所干扰，多个并发事务之间要相互隔离。故 C 项正确。

    A 项错误，原子性是指事务包含的所有操作要么全部成功，要么全部失败回滚。

    B 项错误，一致性是指事务必须使数据库从一个一致性状态变换到另一个一致性状态，也就是说一个事务执行之前和执行之后都必须处于一致性状态。

    C 项正确，隔离性是当多个用户并发访问数据库时，比如操作同一张表时，数据库为每一个用户开启的事务，不能被其他事务的操作所干扰，多个并发事务之间要相互隔离。

    D 项错误，持久性是指一个事务一旦被提交了，那么对数据库中的数据的改变就是永久性的，即便是在数据库系统遇到故障的情况下也不会丢失提交事务的操作。

    故正确答案为 C。

    C 过载绑定

    D 强制绑定
    解析
    本题考查的是程序设计语言中的绑定。 面向对象有三种特性：派生，继承，多态。 其中多态性有两种，一种是依靠函数重载实现的静态绑定，一种是依靠虚函数实现的动态绑定。动态绑定是在运行时进行的，即一个给定的过程调用和响应调用需执行的代码的结合直到调用发生时才进行。这种机制实现了方法的定义与具体的对象无关，而对方法的调用则可以关联于具体的对象。故 B 项正确。

    B 项正确，动态绑定是面向对象程序设计语言中的一种机制，绑定是在运行时进行的，即一个给定的过程调用和响应调用需执行的代码的结合直到调用发生时才进行，这种机制实现了方法的定义与具体的对象无关，而对方法的调用则可以关联于具体的对象。

    C、D 项错误，绑定分为静态绑定（前期绑定）和动态绑定（后期绑定），不存在过载绑定与强制绑定。

    故正确答案为 B

  4. 在 Java 中，负责对字节代码解释执行的是（ ）。

    A 垃圾回收器

    B 虚拟机

    C 应用服务器

    D 编译器

    正确答案是： B

    解析

    本题考查的是 Java 虚拟机的相关知识。java 源代码由编译器编译成字节码存放在 class 文件中。执行时，由 java 虚拟机 (jvm) 读取字节码文件翻译并执行。故 B 项正确。

    A 项错误，在 Java 中，垃圾回收是一个自动的进程可以替代程序员进行内存的分配与回收这些复杂的工作，Java 有四种类型的垃圾回收器。

    B 项正确，Java 字节码文件由虚拟机负责解释执行，从而达到跨平台的目的。

    C 项错误，Java 服务器主要为应用程序提供运行环境，为组件提供服务。从功能上分为两类：WEB 应用服务器和 Java EE 应用服务器。

    D 项错误，Java 编译器的作用是将 java 源程序编译成字节码文件，是最基本的开发工具。

    故正确答案为 B。

  5. 在 Java 编程语言中，异常处理的一般步骤不包括（ ）。

    A 异常捕获

    B 异常抛出

    C 异常处理

    D 异常等待

    正确答案是： D

    解析

    本题考查了 Java 语言。

    在 Java 中，异常处理的方法有两种，

    一种方法是 Java 的异常处理是通过 3 个关键词来实现的：try-catch-finally。使用 try…catch…finally 结构对异常进行捕获和处理；

    另一种方法是通过 throws 和 throw 抛出异常。

    A 项、C 项正确，Java 的异常处理是通过 3 个关键词来实现的：try-catch-finally。用 try 来执行一段程序，如果出现异常，系统抛出（throws）一个异常，可以通过它的类型来捕捉（catch）并处理它，最后一步是通过 finally 语句为异常处理提供一个统一的出口，finally 所指定的代码都要被执行。

    B 项正确，throws 子句中可以同时指明多个异常，说明该方法将不对这些异常进行处理，而是声明抛弃它们。

    D 项错误，异常等待不属于 Java 中异常处理的步骤。

    本题为选非题，故正确答案为 D。

  6. 下面哪个不是 JAVA 关键字？（ ）

    A.integer

    B.double

    C.float

    D.default

    正确答案是： A

    解析

    本题考查计算机语言。

    Java 关键字是电脑语言里事先定义的，有特别意义的标识符，有时又叫保留字，还有特别意义的变量。Java 的关键字对 Java 的编译器有特殊的意义，他们用来表示一种数据类型，或者表示程序的结构等，关键字不能用作变量名、方法名、类名、包名和参数。

    A 项错误，integer，整数 / 整型数，与小数 / 浮点数相对，是编程语言的基本数据类型之一，用以指示变量的数据类型，有时也用于常量（变量、表达式）的强制数据类型转换。

    B 项正确，double 是 Java 关键字，基本数据类型之一，双精度浮点数类型。

    C 项正确，float 是 Java 关键字，基本数据类型之一，单精度浮点数类型。

    D 项正确，default 是 Java 关键字，默认，例如，用在 switch 语句中，表明一个默认的分支。Java8 中也作用于声明接口函数的默认实现。

    本题为选非题，故正确答案为 A。

  7. 欲构造 ArrayList 类的一个实例，此类继承了 List 接口，下列方法正确的是（ ）。

    A. ArrayList myList = new Object();

    B. List myList = new ArrayList();

    C. ArrayList myList = new List();

    D. List myList = new List();

    正确答案是： B

    解析

    本题考查计算机语言。

    A 项错误，只是实例化了一个 ArrayList 类的对象 myList，而没有实现接口。

    B 项正确，不能直接建立接口的实例，只能通过实现接口的相应的类来实现接口。用 List 来声明 MyList 实现接口，并用 ArrayList 构造出一个实例。

    C 项错误，List 并不是一个类，不能用 new 关键字。

    D 项错误，接口不能实例化一个接口。

    故正确答案为 B。

  8. 以下有关选择一种编程语言的理论标准表述中，不正确的是（ ）。

    A 必须是多种语言的界面环境

    B 编译程序发现程序错误的能力强

    C 便于调试和提高软件可靠性

    D 有良好的独立编译机制

    正确答案是： AD

    本题考查的是选择编程语言的标准。

    A 项错误，大多数编程语言的环境为英语，并非多种语言界面。

    B 项正确，编辑程序发现错误多可以提高编程的效率和质量。

    C 项正确，便于调试可以更方便发现编程中存在问题，提高编程效率。

    D 项错误，编译机制应该是跨平台兼容性的，这样才能提高编程的适配性和通用性。

    本题为选非题，故正确答案为 AD。

  9. 下列关于 Java 与 C++ 语言的对比，正确的有（ ）。

    A.C++ 语言丢了指针的概念，引入了真正的数组，在一定程度上减少了不安全问题的发生

    B.Java 比 C++ 语言的可靠性更高

    C.Java 的所有函数和变量都是类的一部分，对象将数据和方法结合起来封装在类中，而 C++ 语言允许将函数和变量定义为全局的

    D.Java 也支持 C++ 语言中的强制类型转换

    正确答案是： BC

    解析

    A 项错误，指针是 C++ 的优势，可以对内存直接操作，当然可能会带来危险。

    B 项正确，Java 丢弃了指针，自动回收垃圾，可靠性更高。

    C 项正确，Java 的变量必须定义在类中，C++ 允许使用 extern 关键字标识变量

    D 项错误，Java 仅支持 C style 的强制类型转换，C++ 除了 C style 还有其他四种强制类型转换。

    正确答案选 BC。
